#  E-Commerce Analytics Project

This project analyzes product profitability and business performance using Python, Machine Learning, and a Streamlit dashboard.

##  Features
- Data cleaning and preprocessing  
- Profitability and margin analysis  
- Top and low-performing products  
- Division-wise performance analysis  
- Pareto (profit concentration) analysis  
- Linear Regression model for profit prediction  
- Interactive Streamlit dashboard  

## 🛠️ Technologies Used
- Python  
- Pandas  
- Matplotlib & Seaborn  
- Scikit-learn  
- Streamlit  

##  Files
- `ML_E_Commerce_Analytics.ipynb` → Data analysis & ML  
- `app.py` → Streamlit dashboard  
- `Nassau Candy Distributor.csv` → Dataset  

##  How to Run

### Run Notebook
Open `.ipynb` file in Jupyter Notebook or VS Code

### Run Dashboard
```bash
streamlit run app.py
